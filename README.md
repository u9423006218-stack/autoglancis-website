# AutoGlancis Website

Mobile car cleaning and detailing service in Ventspils, Latvia.

## Live Demo

[View Website](https://your-username.github.io/autoglancis_website)

## Features

- Modern, responsive design
- Service listings with pricing
- Direct booking integration via [Cal.com](https://cal.com/auto-glancis-tz36dk)
- Instagram Reels embedded in the gallery
- Ventspils map integration

## Tech Stack

- HTML5 + Tailwind CSS
- Vanilla JavaScript
- Static site (no backend)

## Local Development

Simply open `index.html` in your browser.

## Deployment

This site is deployed via GitHub Pages.
